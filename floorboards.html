Upload a room file. At minimum this is a JSON file containing the layout of the room (an array of vertices). You can save a layout to a room file, and reload it later.

Describe the planks you will be using. Dimensions are all in cm. Cut thickness is the thickness of a saw cut. Wherever possible, planks shorter than Min plank length will not be used (it's unavoidable sometimes).

You can start the planking at an offset from the left edge of the room. A positive value will leave a gap between the wall and the first plank. A negative value will mean you'll have to trim the first row of planks lengthways. The offset should never be more than the width of a plank.

0 planks are needed. There will be 0 cuts, and a total wastage of 0cm. "<" planks have their cut edges at the bottom, ">" planks at the top. If you don't like the layout you can the columns.
